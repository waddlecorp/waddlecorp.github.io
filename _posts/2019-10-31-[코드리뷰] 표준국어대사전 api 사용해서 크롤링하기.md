##### 2. 파이썬으로 크롤링을 시작해보자  
먼저, 크롤링을 하기 위한 라이브러리를 설치해야한다.   
urlopen과, BeautifulSoup을 사용하자.   
처음 사용하시는 분은 pip이나 easy-install 등을 이용해 먼저 컴퓨터에 설치하자   


```
from urllib.request import urlopen   
from bs4 import BeautifulSoup
```
